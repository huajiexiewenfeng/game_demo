# 传奇-Demo (Legend of Mir Demo)

这是一个基于 Godot 4 (C#) 开发的 2D 俯视角 ARPG 游戏 Demo，核心机制致敬了经典的《热血传奇》。

## 核心系统与功能

### 1. 角色属性与养成 (Player stats & Levelling)
- **等级与经验 (Leveling System)**：怪物死亡后会掉落经验，玩家获得经验可以升级。每升一级都会提升最大生命值、基础攻击力，并获得 **1点技能点**。
- **动态属性计算**：角色的当前总攻击力基于“基础攻击力 + 武器装备加成 + 被动技能加成”实时计算。

### 2. 战斗系统 (Combat System)
- **目标锁定与锁敌**：鼠标左键点击怪物可将其设为目标，角色会自动寻路走到怪物的攻击范围内进行自动攻击。点击空地则会取消目标并移动。
- **攻击判定与冷却**：基于一定的攻击距离（`AttackRange`）与攻击间隔（`AttackCooldown`）每秒自动挥剑。
- **生命值流失与死亡**：角色和怪物均有生命值，受到攻击时会飘出带有颜色区分的伤害数字（Damage Text）。死亡后怪物会消失并触发掉落。

### 3. 物品与背包系统 (Inventory & Equipment)
- **经典物品图鉴 (Item Database)**：
  - 武器（Weapon）：木剑、乌木剑、青铜剑、屠龙剑等。
  - 防具（Armor）：布衣、皮甲、神圣铠甲等。
  - 首饰（Helmet/Necklace/Bracelet/Ring）：头盔、项链、手镯、戒指等。
  - 消耗品（Consumable）：金疮药（小/中/大）、复活草等。
- **随机掉落机制 (Drop Table)**：怪物死亡时有 50% 概率根据物品的权重（Weight）随机掉落物品，权重越高的低级装备越容易掉落，顶级装备（如屠龙剑）掉率极低。
- **自动拾取与穿戴**：掉落物品会自动进入玩家背包。如果是攻击力更高的武器，玩家会自动穿戴更新。
- **背包界面 (B键打开)**：可视化的 Inventory UI，展示当前拾取到的所有物品及其附加属性。

### 4. 技能树系统 (Skill Tree)
- **技能学习 (Learning Skills)**：玩家消耗升级获得的技能点来学习技能。不同技能有对应的等级要求。
- **快捷键释放**：学习后的技能可以绑定到数字键 `1` 到 `5`，通过按下键盘对应数字键释放。
- **三类技能效果**：
  1. **主动伤害**：如“力劈”、“旋风斩”、“屠龙斩”，对锁定的敌人造成百分比倍率（Damage Multiplier）的伤害。
  2. **主动恢复**：如“金疮回气”、“凤凰涅槃”，立即恢复玩家的大量生命值。
  3. **被动加成**：如“战斗意志”、“霸体护甲”，无需释放，学习后永久增加玩家的基础攻击力、防御力或最大生命值。

### 5. 其他细节与表现 (Polish)
- **动画控制器 (CharacterAnimator)**：解耦的动画脚本，控制 Sprite 缩放、变色、弹跳以模拟走路、待机和受击反馈。
- **跟随摄像机**：开启了平滑移动（Position Smoothing）的 2D 摄像机，确保镜头跟随玩家平缓移动。

## 技术栈
- 引擎：Godot Engine v4.x
- 语言：C# (采用 .NET 技术栈)
