# 传奇 Like ARPG 游戏设计文档 (V1.0)

## 一、 核心玩法循环 (Core Loop)
游戏正式从“无尽刷怪”转变为经典的**“主城休整 + 副本闯关打宝”**模式。

### 1. 绝对安全区（主城：比奇省 / 银杏山谷）
- **功能**：玩家默认出生地，禁止战斗，BGM 平和。
- **核心 NPC 系统**：
  - **铁匠**：买卖低级武器，修理武器耐久度。
  - **服装店**：买卖防具、头盔。
  - **药店老板**：购买“金创药”（回血）、“魔法药”（回蓝），出售多余物品换取金币。
  - **老兵 / 传送阵**：副本入口（走到旁边点击或触碰弹出选关界面）。
- **经济系统**：刷图掉落的杂物或淘汰装备专门带回城卖钱，金币是维持下副本吃药的唯一命脉。

### 2. 副本与关卡梯队 (Dungeon Gradient)
每一关有独立的专属环境和掉落池。初期设定为 4 个阶梯关卡。

**【关卡 1】 新手村外围 (Lv.1 - Lv.7)**
- **生态**：草地、树林。
- **怪物**：鸡、鹿、稻草人。
- **机制**：鸡跑得慢血极少；鹿被攻击后会反击且伤害不低；稻草人防高。
- **掉落**：实体真金币、鸡肉/鹿肉（纯卖钱）、木剑 (攻 2-5)、布衣、小量金创药。
- **关底 Boss**：【变异的稻草人王】（通关击杀后掉落回城卷并开启下一关）。

**【关卡 2】 毒蛇山谷 / 古墓浅层 (Lv.7 - Lv.15)**
- **生态**：山谷、泥地。
- **怪物**：多钩猫、钉耙猫、半兽人。
- **机制**：多钩猫移速快；半兽人血厚攻高。开始出现远程压力。
- **掉落**：铁剑、乌木剑（带魔法属性）、青铜剑、轻型盔甲、传统项链。
- **关底 Boss**：【巨大的毒蛇妖】或【钉耙猫王】。

**【关卡 3】 兽人古墓深处 / 废矿 (Lv.15 - Lv.22)**
- **生态**：黑暗洞穴，火把照明（视野变暗）。
- **怪物**：骷髅、骷髅战将、半兽人掷矛手、半兽人狂战士。
- **机制**：怪物成群结队，骷髅移动慢但攻击力极高，远程掷矛手极度考验拉扯走位。
- **掉落**：八荒剑、斩马刀、中型盔甲、**技能书残页**。
- **关底 Boss**：【骷髅精灵】（浑身血红，移速极快，必须靠剑气卡距离风筝）。

**【关卡 4】 僵尸洞 / 尸王殿 (Lv.22+)**
- **生态**：阴森墓穴。
- **怪物**：普通僵尸、爬行残尸、闪电僵尸。
- **机制**：闪电僵尸会隔空降下雷电极其危险。僵尸死亡后有概率“复活”一次。
- **掉落**：降魔剑、凝霜、**完整高级技能书**。
- **关底 Boss**：【尸王】（经典的锁链声，高防高血，极高爆率）。

## 二、 核心机制设计 (已敲定方案)

### 1. 战利品拾取机制（Looting）
- **自动吸附的爆装大满足体验**：为了兼顾打宝爽感与快节奏体验，怪物死亡后，爆出的金币、低级药水和特殊装备绝不会直接进包，而是化作实体以**“物理爆射溅散动画”**的形式铺满地面。
- 在短暂的停留延迟（约 1 秒体验大爆的震撼）后，满地的战利品会自动受到引力波牵引，化作流光**极速吸入角色体内**提示获取。

### 2. 关卡界面切入方式（Level Entry）
- **NPC 选关大厅**：人物默认常驻于在主城安全区（比奇城）。在安全区中央地带设立【老兵 NPC】。
- 当玩家触碰到老兵交互时，弹出一个极度清晰、高颜值的**【关卡远征大厅界面】**，直接鼠标点击便可读取地图并切入战斗副本。

### 3. 技能学习机制（Skill Progression）
- **保留属性点成长体系**：保留目前极度顺滑好用的升级加点模型。升级提供通用“技能点”，玩家时刻都能凭借等级红利自主构筑打法、决定点开或者升级后续高级技能分支（如精准刺击的后续）。不再拘泥于拼脸爆技能书的繁琐。

## 三、 进阶系统与 UI 交互 (Advanced UI & Systems)
随着游戏复杂度的提升，UI 系统必须由“数据展示”向“沉浸式交互”进化。

### 1. 角色 6 大独立装备槽 (Paper Doll 皮肤系统)
- **面板组成**：摒弃简陋的单武器槽，引入经典的 6 件套网格换装系统：`【武器】`、`【盔甲】`、`【头盔】`、`【项链】`、`【手镯】`、`【戒指】`。
- **属性联动计算**：玩家的最终战斗力（`FinalAttackPower`、`FinalDefense`、`FinalMaxHp`）将由人物肉体裸装基础属性与这 6 个槽位内的神装属性在系统底层（`Player.cs`）精密相加融合而成。
- **呼出方式**：键盘按 `C`（Character）随时开关。

### 2. 背包物品双击交互 (Interactive Inventory)
- 全面淘汰拖拽或单独按钮的冗杂操作，背包系统全面支持 **鼠标左键双击** 极速鉴别：
  - **消耗品（Potion）**：双击金疮药等恢复品，自动计算当前缺损生命值，盈满即止，喝药后背包数量实时递减更新。
  - **穿戴装备（Equipment）**：双击包内兵器/首饰，系统自动校验玩家的 `Level` 境界要求，符合条件立刻完美替换身上所属部位的旧装备退回背包，并光速刷新人物全属性。

### 3. 自定义法术快捷栏 (Customizable Skill Hotbar)
- **按键绑定机制**：技能面板中，针对每一个已参透的“主动释放型法术”，下方动态生成 `[1] - [5]` 五个快捷槽按钮。点击对应按钮便可将其直接映射至键盘主键区 1~5。
- **底层流转**：摒弃硬编码，键盘按下 1~5 键时，引擎直接在实时维护的哈希字典 `Dictionary<int, SkillData>` 内寻址并释放法术。
- **HUD 同步渲染**：底部常驻的极简动作条 UI 会自动捕捉哈希字典，渲染对应的法术名称、高亮状态以及精准计算的冷却覆盖遮罩（Cooldown Mask）。

### 4. 高压割草战斗生态位 (Combat & Density Tuning)
为了确保 ARPG “刷” 的快感满足硬核玩家，战斗生态设定为极高密度的“类吸血鬼幸存者/暗黑破坏神”模式：
- **暴兵体系**：副本切入即达到几十只怪物的满屏承载量，刷新间隔极短（约 0.2 秒），极大化提升 AOE 技能（如旋风斩）的回报。
- **防御收益显著化**：怪物的攻击必定触发真实计算，受击不仅有飙血反馈，怪物伤害会强制与角色的 `FinalDefense`（护甲值）相减。这使得打造极品防具变得极具性价比。
- **经济减压**：药水爆率被大幅度精简压制（上限低于 40%），免除了频繁清理背包满药的痛苦，使得出金、爆神装时的视线瞬间更加纯粹震撼。
- **紧急撤离点**：游戏屏幕右上角悬挂紫色的 `回城 (SafeZone)` 按键，允许在狂暴怪群中支持随时点击脱离位面。
